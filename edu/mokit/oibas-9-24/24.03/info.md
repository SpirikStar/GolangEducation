Создайте массив configs

Массив хранит настройки: Dark: true || false; textSize: uint 
1. Создайте функцию, которая будет добавлять в массив новую настройку (используя сруктуру). 
2. Создайте функцию, которая будет показывать последнюю добавленную настройку которая хранится в массиве. 
3. Создайте функцию, которая будет показывать предпоследнюю добавленную настройку которая хранится в массиве.
