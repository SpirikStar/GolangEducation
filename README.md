### Установка и настройка среды разработки для Go

1. **Установите редактор Visual Studio Code**  
   Перейдите на официальный сайт [https://code.visualstudio.com/](https://code.visualstudio.com/) и скачайте установщик, соответствующий вашей операционной системе. Выполните установку.

2. **Скачайте и установите Go**  
   Перейдите на официальный сайт языка Go: [https://go.dev/](https://go.dev/).  
   Откройте раздел **Download** ([https://go.dev/dl/](https://go.dev/dl/)) и загрузите установочный пакет для вашей платформы. Следуйте инструкциям установщика.

3. **Установите расширение Go для VS Code**  
   Запустите Visual Studio Code.  
   Перейдите в Marketplace по ссылке:  
   [https://marketplace.visualstudio.com/items?itemName=golang.Go](https://marketplace.visualstudio.com/items?itemName=golang.Go)  
   Нажмите **Install**, чтобы добавить официальное расширение Go от команды Google. Оно обеспечит поддержку IntelliSense, навигации по коду, отладки, форматирования и других функций разработки на Go.

После выполнения этих шагов ваша среда будет готова к разработке приложений на языке Go.
