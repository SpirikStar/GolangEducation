### Установка и настройка среды разработки для Go ()

1. **Установите редактор Visual Studio Code**  
   Перейдите на официальный сайт [https://code.visualstudio.com/](https://code.visualstudio.com/) и скачайте установщик, соответствующий вашей операционной системе. Выполните установку.

2. **Скачайте и установите Go**  
   Перейдите на официальный сайт языка Go: [https://go.dev/](https://go.dev/).  
   Откройте раздел **Download** ([https://go.dev/dl/](https://go.dev/dl/)) и загрузите установочный пакет для вашей платформы. Следуйте инструкциям установщика.

3. **Установите расширение Go для VS Code**  
   Запустите Visual Studio Code.  
   Перейдите в Marketplace по ссылке:  
   [https://marketplace.visualstudio.com/items?itemName=golang.Go](https://marketplace.visualstudio.com/items?itemName=golang.Go)  
   Нажмите **Install**, чтобы добавить официальное расширение Go от команды Google. Оно обеспечит поддержку IntelliSense, навигации по коду, отладки, форматирования и других функций разработки на Go.

После выполнения этих шагов ваша среда будет готова к разработке приложений на языке Go.

### Установка Go и настройка VS Code (для Windows)
1. **Установите расширение из `.vsix`-файла**  
   Откройте **PowerShell** (нажмите `Win + X` → выберите *Windows PowerShell*).  
   Выполните команду, указав **полный путь к вашему `.vsix`-файлу**:

   ```powershell
   code --install-extension "C:\путь\к\вашему\файлу.vsix"
   ```

   Например:
   ```powershell
   code --install-extension "C:\Users\ВашеИмя\Downloads\go-extension.vsix"
   ```
2. **Распакуйте архив**  
   Распакуйте файл `go1.25.6.windows-amd64.zip` в любую удобную папку на вашем компьютере. Например:  
   `C:\Users\ВашеИмя\Desktop\go`

3. **Откройте настройки VS Code**  
   Запустите VS Code → нажмите `Ctrl + ,` (или зайдите через меню: **File → Preferences → Settings**).

4. **Найдите настройки Go**  
   В строке поиска вверху введите:  
   ```
   go: goroot
   ```

5. **Откройте файл `settings.json`**  
   Справа вверху нажмите на значок «{}» («Open Settings (JSON)»).  
   Добавьте туда следующие строки, **подставив свой реальный путь** к распакованной папке Go:

   ```json
   "go.goroot": "C:\\Users\\ВашеИмя\\Desktop\\go",
   "go.gobin": "C:\\Users\\ВашеИмя\\Desktop\\go\\bin"
   ```

   Обратите внимание: обратные слэши `\` нужно удваивать — пишите `\\`.

6. **Создайте файл программы**  
   В папке, где вы работаете (например, в той же папке с Go или в отдельной рабочей директории), создайте новый файл с именем `main.go`.  
   Скопируйте в него следующий код:

   ```go
   package main

   import "fmt"

   func main() {
       fmt.Println("Проверка...")
   }
   ```

   Убедитесь, что имя файла именно `main.go` — без опечаток и с расширением `.go`.

7. **Запустите программу**  
   Откройте терминал (в VS Code: `Ctrl + ~` или через меню **Terminal → New Terminal**).  
   Убедитесь, что вы находитесь в той же папке, где лежит файл `main.go`. Проверить содержимое папки можно командой:

   ```powershell
   ls
   ```

   Если в списке вы видите `main.go` — всё в порядке.  
   Теперь запустите программу командой:

   ```powershell
   go run main.go
   ```

   Вы должны увидеть в терминале:

   ```
   Проверка...
   ```
