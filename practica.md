## Блок 1 — Базовый синтаксис и переменные

**1.** Объявите переменную `name` типа `string` со значением `"Alice"` и выведите её на экран.

**2.** Объявите три переменные в одном блоке `var`: `x` равно `10`, `y` равно `20`, `z` равно `30`. Выведите их сумму.

**3.** Используя короткое объявление `:=`, создайте переменную `pi` равную `3.14` и выведите её тип через `fmt.Printf`.

**4.** Объявите константу `MaxSize` равную `100` и выведите её значение.

**5.** Объявите несколько констант через `iota`: `Sunday`, `Monday`, `Tuesday`. Выведите их числовые значения.

---

## Блок 2 — Условия и циклы

**6.** Напишите программу, которая проверяет: если число `n` больше нуля — выводит `"positive"`, если меньше — `"negative"`, иначе `"zero"`.

**7.** Напишите цикл `for`, который выводит числа от 1 до 10.

**8.** Напишите цикл `for` в стиле `while` — повторяйте удвоение числа `n = 1` до тех пор, пока `n` не превысит 100. Выведите результат.

**9.** Напишите бесконечный цикл с `break`: выводите числа от 1 до 5, затем выходите.

**10.** Напишите цикл, который выводит только чётные числа от 1 до 20 с помощью `continue`.

---

## Блок 3 — Функции

**11.** Напишите функцию `add(a, b int) int`, которая возвращает сумму двух чисел.

**12.** Напишите функцию `swap(a, b string) (string, string)`, которая возвращает два значения в обратном порядке.

**13.** Напишите функцию `factorial(n int) int`, которая рекурсивно вычисляет факториал.

**14.** Напишите функцию с переменным числом аргументов `sum(nums ...int) int`, которая возвращает их сумму.

**15.** Напишите функцию, которая возвращает другую функцию — замыкание-счётчик: каждый вызов увеличивает счётчик на 1.

## Блок 4 — Массивы и срезы

**16.** Объявите массив из 5 целых чисел, заполните его вручную и выведите каждый элемент через цикл `for range`.

**17.** Создайте срез `[]int` с числами от 1 до 5, добавьте элемент `6` через `append` и выведите результат.

**18.** Создайте срез из 10 нулей с помощью `make`, затем присвойте каждому элементу его индекс.

**19.** Напишите функцию, которая принимает срез целых чисел и возвращает новый срез только с чётными числами.

**20.** Скопируйте срез `a` в срез `b` с помощью функции `copy` и убедитесь, что изменение `b` не влияет на `a`.

## Блок 5 — Карты (map)

**21.** Создайте карту `map[string]int`, где ключи — имена студентов, значения — их оценки. Выведите все пары ключ-значение.

**22.** Проверьте, существует ли ключ `"Bob"` в карте. Если нет — добавьте его со значением `5`.

**23.** Удалите элемент из карты по ключу и убедитесь, что он удалён.

**24.** Напишите функцию, которая принимает срез строк и возвращает карту, где ключ — строка, значение — её длина.

**25.** Подсчитайте количество вхождений каждого слова в строке `"apple banana apple cherry banana apple"` используя карту.

## Блок 6 — Структуры

**26.** Объявите структуру `Person` с полями `Name string` и `Age int`. Создайте экземпляр и выведите его.

**27.** Напишите метод `Greet()` для структуры `Person`, который выводит `"Hello, my name is <Name>"`.

**28.** Объявите структуру `Rectangle` с полями `Width` и `Height`. Добавьте метод `Area() float64`, возвращающий площадь.

**29.** Реализуйте встраивание структур: создайте `Animal` с полем `Name`, и `Dog`, которая встраивает `Animal` и добавляет поле `Breed`.

**30.** Создайте срез из трёх структур `Person`, отсортируйте его по полю `Age` с помощью `sort.Slice`.

## Блок 7 — Интерфейсы

**31.** Объявите интерфейс `Shape` с методом `Area() float64`. Реализуйте его для структур `Circle` и `Rectangle`.

**32.** Напишите функцию `printArea(s Shape)`, которая принимает интерфейс `Shape` и выводит площадь фигуры.

**33.** Используйте пустой интерфейс `interface{}`: напишите функцию `describe(i interface{})`, которая выводит тип и значение аргумента через `fmt.Printf("%T %v")`.

**34.** Сделайте type assertion: получите конкретный тип из переменной `interface{}` и обработайте случай ошибки (запятая-ок идиома).

**35.** Реализуйте интерфейс `fmt.Stringer` для структуры `Person` — метод `String() string`, возвращающий `"Name (Age)"`.

## Блок 8 — Ошибки

**39.** Напишите функцию, которая читает файл и возвращает его содержимое. Обработайте ошибку открытия файла.

## Блок 9 — Горутины и каналы

**41.** Запустите горутину, которая выводит `"Hello from goroutine"`. Используйте `time.Sleep` в main, чтобы дождаться её.

**42.** Создайте небуферизованный канал `chan int`. Отправьте значение из горутины, получите его в main и выведите.

**43.** Создайте буферизованный канал на 3 элемента. Отправьте три значения без горутины, затем прочитайте их в цикле.

**44.** Используйте `WaitGroup` из пакета `sync`: запустите 5 горутин, каждая выводит свой номер, дождитесь завершения всех.

**45.** Напишите конвейер из двух горутин: первая генерирует числа от 1 до 5 в канал, вторая читает из него и выводит квадраты.

## Блок 10 — Стандартная библиотека и разное

**47.** Используя `strconv`, преобразуйте строку `"42"` в `int`, прибавьте `8` и преобразуйте результат обратно в строку.

**48.** Используя `time`, выведите текущую дату и время в формате `"2006-01-02 15:04:05"` (Go-формат).

**49.** Напишите функцию `isPalindrome(s string) bool`, которая проверяет, является ли строка палиндромом. Используйте `[]rune` для корректной работы с Unicode.

**50.** Реализуйте простой стек на основе среза: структура `Stack` с методами `Push(v int)`, `Pop() (int, bool)` и `IsEmpty() bool`.
